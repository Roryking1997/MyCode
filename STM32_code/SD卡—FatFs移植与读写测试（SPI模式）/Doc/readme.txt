【*】程序简介

-工程名称：SDCARD―FatFs移植与读写测试
-实验平台: 野火STM32 MINI 开发板 
-MDK版本：5.16
-ST固件库版本：3.5

【 ！】功能简介：
使用文件系统对SD卡进行读写

学习目的：学习STM32如何移植文件系统到SPI SD卡


【 ！】实验操作：
在开始进行SD卡基本测试前，请给开发板插入32G以内的SD卡。		

电脑端使用串口调试助手，选择电脑与STM32相连的COM口，设置为115200-N-8-1，
复位开发板，即可接收STM32串口发送给电脑的调试信息。	

/***************************************************************************************************************/

【*】 引脚分配

(microSD卡座，支持32G以内的SD卡)：
microtSD卡座的接口与STM32的SPI相连。

		(CLK)SCK	<--->PA5
		(DATA0)MISO	<--->PA6
		(CMD)MOSI	<--->PA7
		(DATA3)CS	<--->PA8

串口(TTL-USB TO USART)：
CH340的收发引脚与STM32的发收引脚相连。
	RX<--->PA9
	TX<--->PA10
